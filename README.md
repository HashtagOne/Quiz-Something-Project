# Python Quiz App

Currently, a command-line quiz application built in Python. Features a two-layer topic navigation system, per-user JSON leaderboard, input validation, and randomized questions.

## Features

- Typewriter effect for all text output
- Two-layer topic and subtopic navigation
- Randomized question order per session
- Input validation for name and age credentials
- Multiple accepted answers per question
- Score tracking with pass/fail messaging
- Per-user JSON leaderboard saved to disk
- Leaderboard filtered and sorted by topic

## Topics Available

- Science
  - Biology
  - Geography
- Social Sciences
  - Geopolitics

## How to Run

```
python Quiz.py
```

Requires Python 3. No external dependencies.

## Built With

- Python
- JSON
- os / time / random standard libraries
